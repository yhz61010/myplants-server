FERNS INDOOR

银脉凤尾蕨

Pteris ensiformis 'Evergemiensis'

psychiatry Care Requirements

water_drop Watering

Keep soil consistently moist. Do not let dry out completely.

light_mode Light

Medium to bright indirect light. Avoid direct sun.

养护指南

银脉凤尾蕨喜欢温暖湿润的环境。在春夏季节，保持盆土微湿，但避免积水。经常向叶面喷水以增加空气湿度。冬季减少浇水，保持稍微干燥。每月施一次稀薄的液肥即可。注意避免强光直射，以免叶片干枯。
